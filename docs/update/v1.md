> 更新时间：2020.7.14

❗ 下面是框架升级需要手动进行的变更：

- 新版本由于引进了自定义 Swoole Server 事件的机制，对 global.php 全局配置文件有了变动，需要添加以下内容才能正常启动（旧版本升级新版本用户，新用户无需操作）：

```php
/** 注册 Swoole Server 事件注解的类列表 */
$config['server_event_handler_class'] = [
    \Framework\ServerEventHandler::class, //默认不可删除，否则会不能使用框架
];
```
